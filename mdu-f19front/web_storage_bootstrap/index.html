HTML5 Web Storage

HTML web storage; better than cookies.

HTML web storage provides two objects for storing data on the client: localStorage and sessionStorage.

Learn more at W3Schools.com

Local Storage

... stores data with no expiration date. The data will not be deleted when the browser is closed and will be available next time you open the browser.

Session Storage

... stores data for one session. The data is lost when the browser tab is closed and will NOT be available next time you open the browser.
